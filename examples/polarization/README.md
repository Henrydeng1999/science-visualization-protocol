# 示例：偏振

> 关联基本量：**发光强度 cd**

## 科学概念

偏振片只允许特定方向振动的光通过。透过强度近似满足 I=I₀cos²θ。

## 可调参数

- 偏振片角度
- 入射强度

## 观察目标

拖动参数，观察画面中的图形、读数或分布如何变化，并总结参数变化和物理现象之间的关系。

## 实现说明

本示例为教学可视化模型，优先展示趋势和因果关系。部分数值采用相对尺度或夸张比例，不作为精密仿真。

## 运行方式

复制 [sketch.js](sketch.js) 到 [p5.js Web Editor](https://editor.p5js.org/) 运行，或在本地用浏览器打开本目录的 `index.html`。
