# 示例 5：黑体辐射与普朗克

> 关联基本量：**热力学温度 K** ｜ 经典 → 量子路径上的第一站

## 科学概念

任何有温度的物体都会向外辐射电磁波。把它的辐射强度按波长画出来，就是一条「黑体辐射曲线」。温度改变时：

- **峰值波长左移**（维恩位移定律 λ_max·T ≈ 2.898×10⁶ nm·K）：温度越高，峰值越偏向短波（蓝），物体颜色从暗红 → 橙 → 黄白 → 蓝白。
- **总辐射急剧增大**（斯特藩–玻尔兹曼定律，总功率 ∝ T⁴）：温度翻倍，总辐射变成 16 倍。

## 为什么是量子的起点

19 世纪末，经典物理预言短波辐射会趋于无穷（「紫外灾难」），与实验曲线完全不符。1900 年普朗克假设振子的能量只能取 hν 的整数倍——能量是**一份份（量子化）**地发出，才首次推出与实验吻合的公式。这条曲线因此被视为量子物理的开端。

普朗克公式（本示例用其相对形式）：

```
B(λ, T) ∝ 1/λ⁵ · 1/( e^(hc/λkT) − 1 )
```

## 可调参数

- 温度 T（1000–12000 K）
