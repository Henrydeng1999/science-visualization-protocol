# 示例 6：光是一种波

> 关联基本量：**时间 s、长度 m** ｜ 经典光学基础

## 科学概念

光是一种电磁波。为了先理解「波」本身，本示例用一列横波演示三个基础量：

- **波长 λ**：相邻两个波峰之间的距离。
- **频率 f**：单位时间内振动的次数。
- **振幅 A**：偏离平衡位置的最大距离，表示振动强弱。

波速、波长和频率满足：

```text
v = λf
```

## 可调参数

- 波长 λ
- 频率 f
- 振幅 A
- 是否显示波长标尺

## 观察目标

调短波长时，波峰变密；调高频率时，波传播与振动更快；调大振幅时，波峰和波谷离中心线更远。这个基础页面为后续「波长决定颜色」「干涉」「双缝」做准备。
