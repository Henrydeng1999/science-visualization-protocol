# 示例 4：不同形状磁铁的磁场

## 科学概念

磁铁周围存在磁场，可以用磁感线（磁力线）描述。磁感线从 N 极发出、回到 S 极，方向就是该点小磁针 N 极的指向；磁感线越密的地方磁场越强，且任意两条磁感线都不会相交。

磁铁的**形状**决定了 N、S 两极的位置，因而改变磁场的分布：

- **条形磁铁**：两极在两端，磁感线对称地向四周展开。
- **U 形（马蹄）磁铁**：两极靠近且朝同一方向，磁感线集中在两极之间的开口处，磁场更强、更均匀。
- **圆形磁铁**：横向磁化时，左右成为 S、N 两极，磁场形似一个短的条形磁铁。

## 可调参数

- 磁铁形状（下拉选择：条形 / U 形 / 圆形）
- 磁感线数量（滑块）
- 探测点位置（移动鼠标，用小磁针读出该点磁场方向）

## 实现说明

把每个磁极近似为一个「磁荷」点源（N 极为正、S 极为负），某点磁场由各磁荷的场叠加而成。磁感线通过从 N 极附近出发、沿磁场方向逐步积分（field line tracing）绘制，到达 S 极或离开画布时停止。
