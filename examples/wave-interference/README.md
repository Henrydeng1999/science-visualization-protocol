# 示例 8：两束光相遇：干涉

> 关联基本量：**时间 s、长度 m** ｜ 双缝干涉的前置概念

## 科学概念

波相遇时会叠加。两列光波如果相位相同，波峰遇到波峰，会发生**相长干涉**；如果相位相差 180°，波峰遇到波谷，会发生**相消干涉**。

这正是双缝实验中明暗条纹的基础：屏幕上有些位置两路光程差让光波同相，因此变亮；有些位置让光波反相，因此变暗。

## 可调参数

- 相位差 Δφ
- 波长 λ
- 振幅 A
- 是否显示两列原始波

## 观察目标

拖动相位差滑块，观察合成波如何从增强逐渐变为抵消。理解「光会干涉」是证明光具有波动性的关键证据。
