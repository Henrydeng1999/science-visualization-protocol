# 量子重力梯度仪原理

这个实验用同一台地表仪器内部的上下两团冷原子，模拟量子重力梯度仪的差分测量。地下高密度异常体对两团原子产生略有不同的附加引力加速度，仪器读取的是这个差值，而不是单点重力值。

## 可调参数

- 地下异常体质量：控制异常体引起的附加引力强弱。
- 异常体深度：控制异常体到仪器的距离。
- 原子云基线：控制两团冷原子之间的距离。
- 测线位置：模拟仪器沿地表扫测。

## 教学重点

- 重力梯度关注的是空间变化率：`梯度 ≈ Δg / L`。
- 差分测量能突出局部异常，并削弱共同背景信号。
- 异常体越近、质量越大，梯度信号越强。
- 地下的是被探测的密度异常，不是传感器；两团原子都在地表设备内部。
