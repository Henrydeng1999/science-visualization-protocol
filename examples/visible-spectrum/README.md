# 示例 7：波长与颜色

> 关联基本量：**长度 m、时间 s、发光强度 cd** ｜ 彩虹色散到光电效应的桥

## 科学概念

可见光大约位于 380-750 nm。不同波长的光被眼睛感受为不同颜色：

- 红光波长较长，频率较低。
- 紫光波长较短，频率较高。
- 光子能量满足 `E = hf`，所以频率越高，单个光子的能量越高。

这个示例把「颜色」「波长」「频率」「光子能量」放在同一张画面里，帮助学生从彩虹自然走向光电效应。

## 可调参数

- 光的波长 λ（380-750 nm）

## 观察目标

移动波长滑块，观察颜色、波形疏密、频率和单个光子能量如何一起变化。重点理解：更亮不等于单个光子能量更高，单个光子的能量主要由频率决定。
