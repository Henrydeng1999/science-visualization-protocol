# 示例 9：双缝干涉

> 关联基本量：**时间 s、长度 m、发光强度 cd** ｜ 经典光学到量子双缝的桥

## 科学概念

光通过两条狭缝后，两条缝可以看成两个同步波源。到达屏幕上某一点时，两路光的路程差决定它们是相长还是相消：

- 路程差为整数个波长时，形成亮纹。
- 路程差为半个波长的奇数倍时，形成暗纹。

亮纹间距近似满足：

```text
Δy ≈ λL / d
```

其中 `λ` 是波长，`L` 是屏幕距离，`d` 是双缝间距。

## 可调参数

- 波长 λ
- 双缝间距 d
- 屏幕距离 L
- 连续光 / 单光子累积模式
- 是否显示波前

## 观察目标

增大波长或屏幕距离，条纹间距会变大；增大双缝间距，条纹会变密。切换到单光子模式后，单个落点看似随机，但长期累积仍会显出干涉条纹，这是通向量子双缝实验的重要入口。

## 实现说明

本示例使用简化模型计算屏幕上各点两路光程差，并以 `cos²(phase/2)` 表示干涉强度。单光子模式使用按强度加权的随机采样，表现“单次随机、整体成纹”的现象趋势。
