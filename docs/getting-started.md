# 学生快速入门

## 你需要准备什么

- 一台能上网的电脑或平板。
- 一个想探究的科学问题，例如：
  - 彩虹为什么是弧形的？
  - 把球抛出去，落点由什么决定？
  - 电荷周围为什么会有“力”？

## 四步生成你的科学动画

### 第 1 步：描述问题

用一句话写下你想探究的现象：

> 我想知道：太阳光进入水滴后，为什么能分成七种颜色？

### 第 2 步：填写实验模板

打开 [docs/templates/experiment-template.md](templates/experiment-template.md)，把问题拆成几个小空：

- 科学概念：光的折射、反射、色散
- 要模拟的对象：水滴、太阳光线、不同颜色的光
- 可调参数：入射角度、水滴大小
- 要观察的结果：各色光折射后分开的角度

### 第 3 步：生成 Prompt 并交给 AI

用 [docs/templates/prompt-template.md](templates/prompt-template.md) 把你的内容套成一段 Prompt，发给 AI（Cursor、ChatGPT、Claude 等都可以）。

### 第 4 步：运行代码

1. 打开 [p5.js Web Editor](https://editor.p5js.org/)。
2. 把 AI 生成的代码粘贴进去。
3. 点击播放按钮，拖动滑块，观察变化。

## 遇到问题时怎么办

| 问题 | 解决方式 |
|------|---------|
| 代码报错 | 把报错信息复制给 AI，说“请帮我修复这段 p5.js 代码”。 |
| 动画效果不满意 | 告诉 AI“让颜色更鲜艳”或“加一条光线轨迹”。 |
| 不知道该怎么填模板 | 先问 AI“我想研究 XX，应该怎么拆成科学概念和参数？” |
