# 项目定位：从七个基本物理量到量子现象

## 一句话目标

以 **SI 七个基本物理量** 为线索，用基础实验的可视化模拟，搭一座从「经典直观现象」通向「量子力学 / 量子现象」的台阶，让小初高学生看得懂、玩得动。

## 为什么用「七个基本物理量」串起来

国际单位制（SI）的一切量都由七个基本量导出。它们既是经典物理的地基，也是量子物理的入口（普朗克常数、电子电荷、玻尔兹曼常数等量子常量，正是用这七个基本量重新定义的）。以它们为骨架，能把零散的实验组织成一条从经典到量子的认知路径。

| 基本量 | 单位 | 现有/经典示例 | 通向的量子主题 |
|--------|------|---------------|----------------|
| 时间 t | s | 抛物运动、光波传播、干涉 | 波函数随时间演化、跃迁寿命 |
| 长度 l | m | 彩虹色散、光的波长、波长与颜色 | 德布罗意波长、原子尺度 |
| 质量 m | kg | 抛物运动（重力与质量） | 电子质量、波粒二象性 |
| 电流 I | A | 点电荷电场线、不同形状磁铁的磁场 | 元电荷、量子化电导 |
| 热力学温度 T | K | 黑体辐射与普朗克 ✅ | 普朗克黑体辐射、能量量子化 |
| 物质的量 n | mol | （待补：阿伏伽德罗） | 原子计数、能级布居 |
| 发光强度 Iv | cd | 彩虹 / 光、波长与颜色、光的干涉 | 光电效应、光子能量 E=hν |

> 磁场示例（不同形状磁铁）与电流量 A 同源（运动电荷产生磁场），归在电流主题下。

## 现有示例归类

- **时间 s · 质量 kg** → 抛物运动
- **长度 m · 时间 s · 发光强度 cd** → 光是一种波、波长与颜色、两束光相遇：干涉、彩虹与光的色散
- **电流 A** → 点电荷电场线、不同形状磁铁的磁场
- **热力学温度 K** → 黑体辐射与普朗克（首个量子主题，量子化的起点）

## 量子示例 Roadmap（后续）

按「从经典示例自然延伸」排序，每个都对应一个基本量入口：

1. ~~**黑体辐射与普朗克**（温度 K）~~ ✅ 已完成 — 改变温度看辐射光谱，引出「能量只能一份份地给」。
2. ~~**光是一种波**（时间 s / 长度 m）~~ ✅ 已完成 — 先理解波长、频率、振幅与传播。
3. ~~**波长与颜色**（长度 m / 发光强度 cd）~~ ✅ 已完成 — 连接可见光谱、频率与光子能量 E=hf。
4. ~~**两束光相遇：干涉**（时间 s / 长度 m）~~ ✅ 已完成 — 为双缝实验中的明暗条纹打基础。
5. **光电效应**（发光强度 cd / 光子能量）— 改变光的频率与强度，看电子是否逸出，引出 E=hν。
6. **玻尔原子能级与光谱线**（长度 m / 光）— 电子在能级间跃迁，发出特定波长的光，呼应彩虹的「光谱」。
7. **德布罗意波长**（长度 m · 质量 kg）— 由动量算波长，连接抛物运动里的「质量·速度」。
8. **电子双缝干涉**（综合）— 单个电子逐个打到屏上，累积出干涉条纹，展示波粒二象性。

## 给做新示例的人

- 新增示例时，在 README 标注它主要关联哪个（些）基本物理量，以及它在「经典 → 量子」路径上的位置。
- 保持本仓库一贯做法：清晰度优先于数值精确，可夸大真实参数让现象可见，但保留物理趋势并注明（见彩虹示例的折射率放大）。
