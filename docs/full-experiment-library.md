# 七个基本物理量完整实验库规划

## 目标

以 SI 七个基本物理量为主线，构建一个 35 个交互型小动画组成的科学可视化实验库。每个基本物理量对应 5 个实验，既能独立学习，也能串成一条从经典现象到量子启蒙的完整主线。

## 完整主线

这条主线同时覆盖七个基本物理量，适合作为首页推荐路径、Demo 演示路径和课程化学习路径。

| 顺序 | 基本物理量 | 代表实验 | 学习作用 |
|---|---|---|---|
| 1 | 时间 s | 抛物运动 | 从“随时间变化的运动”进入物理建模。 |
| 2 | 质量 kg | 碰撞与动量守恒 | 理解质量、速度、动量和守恒思想。 |
| 3 | 电流 A | 点电荷的电场线 / 简单电路 | 从电荷、电场过渡到电流和电路。 |
| 4 | 长度 m | 波长与颜色 / 双缝干涉 | 用波长、光程差和尺度理解光学现象。 |
| 5 | 发光强度 cd | 光的色散 / 光电效应 | 从亮度、颜色和光子能量进入光的量子性。 |
| 6 | 热力学温度 K | 黑体辐射 | 从温度和热辐射进入能量量子化。 |
| 7 | 物质的量 mol | 阿伏伽德罗常数 / 能级布居 | 把微观粒子数、统计分布和宏观量联系起来。 |

建议路径：

```text
抛物运动
→ 碰撞与动量守恒
→ 点电荷的电场线
→ 简单电路
→ 波长与颜色
→ 双缝干涉
→ 光电效应
→ 黑体辐射
→ 阿伏伽德罗常数
→ 能级布居
```

## 开发优先级

### 第一批：补齐光学到量子路径

1. 衍射
2. 偏振
3. 光电效应
4. 距离与亮度平方反比

### 第二批：补齐力学与电磁基础

1. 单摆周期
2. 弹簧振子
3. 碰撞与动量守恒
4. 简单电路
5. 通电导线磁场

### 第三批：补齐热学与物质的量

1. 气体分子热运动
2. 热传导
3. 阿伏伽德罗常数
4. 摩尔质量换算
5. 气体粒子数与压强

### 第四批：进入微观与量子深化

1. 德布罗意波长
2. 能级布居
3. 麦克斯韦速度分布
4. 质能关系示意
5. 量子化电导或电子双缝干涉
