# 科学原理可视化 Demo 跟进计划

## 微信群简介文案

这是一个面向中小学生的科学原理可视化互动原型。项目以 SI 七个基本物理量为主线，把时间、长度、质量、电流、温度、物质的量和发光强度串联成一张学习地图。当前 Demo 已完成力学、电场、磁场、光波、光谱、干涉、双缝和黑体辐射等交互页面，学生可以通过拖动滑块直接观察现象变化。当前已按七个基本物理量扩展为 35 个交互小动画，形成一套从经典物理逐步走向量子启蒙的网页互动实验库。

## 当前 Demo 重点

- 首页已经按学习路径组织内容，包含明确的“进入实验”按钮。
- 当前包含 35 个交互实验页面。
- 已部署为静态网页，适合用浏览器直接演示和录屏。
- 技术上采用纯 HTML + CSS + p5.js，后续扩展成本低。

## 演示视频脚本

### 视频 1：首页与学习路径

时长建议：20-30 秒

操作步骤：

1. 打开首页。
2. 停留在标题和简介区域。
3. 鼠标移动到“推荐学习路径”区域。
4. 依次扫过“认识光”“从经典到量子”“场的世界”三张路径卡片。
5. 点击“从这条路径开始”，进入“光是一种波”。

画面重点：

- 让观众看到这不是零散页面，而是按学习路径组织的互动实验库。
- 重点展示入口按钮和路径设计。

### 视频 2：光的波动性链路

时长建议：45-60 秒

操作步骤：

1. 从“光是一种波”页面开始。
2. 拖动“波长 λ”，观察波峰变疏或变密。
3. 拖动“频率 f”，观察波传播速度变化。
4. 点击底部“下一个实验：波长与颜色”。
5. 在“波长与颜色”页面拖动波长，从红光移动到紫光。
6. 点击“下一个实验：两束光相遇：干涉”。
7. 点击“相长”和“相消”按钮，展示合成波变化。

画面重点：

- 波长、频率、颜色、干涉是一条连续学习链。
- 滑块变化要慢一点，每次停留 1-2 秒。

### 视频 3：双缝干涉与量子入口

时长建议：45-60 秒

操作步骤：

1. 打开“双缝干涉”页面。
2. 拖动“波长 λ”，观察条纹间距变化。
3. 拖动“双缝间距 d”，观察条纹变密或变疏。
4. 将“显示模式”切换为“单光子累积”。
5. 停留几秒，让随机光子逐渐累积出条纹。
6. 点击“下一个实验：彩虹与光的色散”或直接切换到“黑体辐射与普朗克”。
7. 在“黑体辐射”页面拖动温度，展示曲线峰值移动。

画面重点：

- 双缝页面负责展示“经典干涉现象如何引出量子问题”。
- 黑体辐射页面负责展示“从经典现象进入量子化概念”。

### 视频 4：场的世界

时长建议：30-45 秒

操作步骤：

1. 从首页点击“进入场的路径”。
2. 在“点电荷的电场线”页面移动鼠标，观察试探电荷受力方向。
3. 点击下一个实验，进入“不同形状磁铁的磁场”。
4. 切换条形磁铁、U 形磁铁、圆形磁铁。
5. 移动鼠标，展示小磁针方向变化。

画面重点：

- 把不可见的“场”变成可观察、可操作的图形。
- 展示电场和磁场可以作为一个连续主题学习。

## 未来内容规划

目标：围绕七个基本物理量，每个方向 5 个交互型演示小动画，形成 35 个基础实验组成的互动实验库。当前清单已全部具备可运行页面，完整清单见 [full-experiment-library.md](full-experiment-library.md)。

| 基本物理量 | 交互演示规划 |
|---|---|
| 时间 s | 抛物运动、单摆周期、弹簧振子、波传播、放射性衰变 |
| 长度 m | 波长与颜色、双缝干涉、衍射、透镜成像、德布罗意波长 |
| 质量 kg | 抛物运动、碰撞与动量守恒、斜面运动、万有引力、质能关系示意 |
| 电流 A | 点电荷的电场线、简单电路、通电导线磁场、螺线管磁场、电磁感应 |
| 热力学温度 K | 气体分子热运动、热传导、相变、麦克斯韦速度分布、黑体辐射 |
| 物质的量 mol | 阿伏伽德罗常数、摩尔质量换算、气体粒子数与压强、化学反应配比、能级布居 |
| 发光强度 cd | 距离与亮度平方反比、光的色散、偏振、光电效应、激光与普通光 |

## 阶段计划

### 第一阶段：Demo 验证

状态：进行中

目标：

- 完成首页、学习路径和首批交互实验。
- 验证网页动画是否能清晰表达科学概念。
- 形成可录屏、可转发、可部署的 Demo。

### 第二阶段：内容扩展

目标：

- 每个基本物理量补齐 3-5 个实验。
- 优先补充：光电效应、衍射、偏振、单摆、碰撞、电磁感应、阿伏伽德罗常数。
- 逐步形成可按年级和主题筛选的实验库。

### 第三阶段：课程化

目标：

- 增加教师任务单、学生实验记录模板、课堂活动路径。
- 每个实验页增加“观察任务”“关键结论”“下一步学习”。
- 支持按课程章节组织演示。

### 第四阶段：AI 辅助生成

目标：

- 允许教师或学生用自然语言描述问题。
- AI 自动拆解科学概念、变量、观察目标。
- 自动生成新的 p5.js 交互实验草稿。
